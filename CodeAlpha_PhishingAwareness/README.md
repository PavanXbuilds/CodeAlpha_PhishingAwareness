**# CodeAlpha Project: Cyber Phishing Awareness \& Prevention System**



**A comprehensive security training project developed during my CodeAlpha Cyber Security Internship. This repository features both an educational blueprint and an interactive evaluation module designed to combat modern social engineering vectors.**



**## 🚀 Project Overview**

**Phishing remains the primary initial access vector for enterprise data breaches. This project delivers a streamlined framework for training end-users to recognize malicious indicators across communication channels.**



**## 📂 Repository Structure**

**\* `/presentation`: Features the complete curriculum covering attack methodologies, domain typosquatting, and prevention guidelines (PPTX \& PDF formats).**

**\* `/quiz`: A lightweight, interactive web component built using clean HTML5, CSS3, and native JavaScript to test user resilience immediately following the presentation.**



**## 🛠️ Deployment \& Execution**

**To execute the interactive evaluation engine locally:**

**1. Clone this repository or download the source files.**

**2. Navigate to the `/quiz` folder.**

**3. Open `index.html` directly in any modern web browser. No complex local server configuration required.**
