## 🚀 Project Overview
🚨 **Phishing remains the #1 initial access vector** for enterprise data breaches.
This project delivers a streamlined framework for training end-users to recognize
malicious indicators across communication channels.

## 📂 Repository Structure

| Folder | Contents | Formats |
|---|---|---|
| `/presentation` | Full curriculum — attack methodologies, typosquatting, prevention guidelines | PPTX, PDF |
| `/quiz` | Lightweight interactive evaluation engine to test user resilience | HTML5, CSS3, JS |

## 🛠️ Deployment & Execution

```bash
# 1. Clone the repository
git clone https://github.com/PavanXbuilds/CodeAlpha_PhishingAwareness

# 2. Navigate to the quiz folder
cd CodeAlpha_PhishingAwareness/quiz

# 3. Open directly in browser — no server needed!
open index.html
```

## ⚠️ Disclaimer
This project is strictly for **educational and awareness purposes**.
All phishing scenarios are simulated and designed to build user resilience.
